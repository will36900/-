# 游戏监测计划

## 当前已导入：MVP 版本

项目路径：

```text
outputs/game-monitor-mvp
```

已实现目标：

- 游戏进程识别
- 实时系统与进程采样
- 最近 5 分钟数据缓存
- 游戏退出检测
- 自动生成诊断报告
- 基础规则分析

## MVP 验收标准

- 可以指定一个游戏进程名开始监控
- 游戏运行时持续输出状态
- 游戏退出后自动生成报告
- 报告中包含采样摘要、异常证据和建议

## 第二阶段

- 接入 PresentMon，采集 FPS、1% Low、0.1% Low、帧时间
- 接入 Windows Event Log，采集崩溃和驱动异常事件
- 接入 LibreHardwareMonitor，采集 GPU 温度、显存、功耗、风扇
- 增加游戏配置档案
- 增加桌面 UI

## 第三阶段

- 崩溃 dump 关联分析
- 驱动版本检测
- 超频/降频识别
- 同硬件同游戏问题库
- AI 诊断摘要
